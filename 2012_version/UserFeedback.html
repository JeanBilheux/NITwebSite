What would you like this facility to do differently

  • Have a concrete cage so beam flux and field of view can be increased as much as possible.
  • I hope the beam line can be equipped with heating and in situ mechanical loading capabilities.
  • Nothing!
  • Maybe have a water fountain somewhere in the area.
  • Longer facility operations, more beam time availability.
  • Have more beam time available to users
  • It would be great to have many meetings with other users to share research ideas.
  • This is a prototype instrument, so clearly it cannot be expected to be very versatile, and these comments are more about what could be improved in future. The imaging beamline needs better sample positioning stages. At present it is very limiting on sample dimensions, given it's obvious potential for moderate sized samples. There is an obvious need for links to (and general instructions for) suitable software tools for image reconstruction and analysis for Users. Accessing the HFIR server to access the data took many steps for myself and for my Sandia collaborator - this should have been more straightforward.
  • It was great.

General Comments

  • The only reason I mark dissatisfied on fraction that the facility operates is because there is a high demand for neutron imaging and few days provided for imaging. However, my main project in developing coded source imaging is beginning to be considered development and may help relieve some stress placed on user imaging time needs.
  • Of course I would like more access to the beam line, so a higher fraction of the year in operation would help that.
  • Obviously I would prefer to see HFIR operating more often, but I fully understand the budgetary constraints.
  • The downtime was frustrating however it was caused by a power outage which was clearly outside the control of the staff we were working with.
  • It is a prototype, so I don't expect flawless operation. However, I would like to see a couple of upgrades to improve the performance of the instrument and simplify data reduction. A sample environment providing precise temperature control would be very useful.
